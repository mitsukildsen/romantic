# Romantic

イチオシのデートプランを投稿・閲覧することができるWebアプリケーション。

## 概要
 
  - アカウント作成必須
  - ログインすることで、投稿閲覧・新規投稿・フォロー・いいねが可能。
  - 「いいね」をつけた投稿一覧を閲覧可能。

## バージョン
　Rails 5.2.4.3
　ruby 2.5.7

## インストール
```
$ git clone https://github.com/mitsukildsen/romantic.git
$ cd Romantic
$ rails db:migrate
$ rails db:seed
$ bundle install
```
