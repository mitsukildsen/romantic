# Romantic

イチオシのデートプランを投稿・閲覧することができるWebアプリケーション。

## 概要
 
  - アカウント作成必須
  - ログインすることで、投稿閲覧と新規投稿、口コミ・レビュー評価が可能。
  - レビュー評価が高い順で表示されるランキング一覧ページを実装。
  - 「いいね」をつけた投稿をマイページから閲覧可能。

## インストール
```
$ git clone 
$ cd Romantic
$ rails db:migrate
$ rails db:seed
$ bundle install
```
